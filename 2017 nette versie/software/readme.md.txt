# Software

## Ontwikkel omgeving:

Dit project kan geopend worden met de standaard Arduino software. Dit moet eenmalig worden ingericht:

- Installeer de Arduino IDE 1.8.3 van www.arduino.cc
- Verbind een Arduino Due via USB, en installeer het software pakket dat de Arduino IDE voorstelt.
- Installeer UTFT library in de Arduino IDE. (zie de UTFT documentatie)
- Installeer DueTimer library in de Arduino IDE.
- Optioneel: open memorysaver.h en zet alles uit behalve SSD1289 om het programma 15kB kleiner te maken.

Als je wil kan je ook een andere ontwikkelomgeving gebruiken, zoals Eclipse.

## Programmeren:

- Open dit programma in de Arduino IDE 
- Selecteer Tools > Board > Arduino Due Programming Port.
- Selecteer Tools > Port >  en kies de COM poort waar Arduino Due (Programming Port) bij staat.
- In Arduino IDE op Upload klikken. (rond knopje met pijltje naar rechts)
